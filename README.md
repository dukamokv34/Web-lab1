## Тема роботи
Розробка статичного інтерфейсу Web-додатка

## Мета роботи
Ознайомитись із засобами мов HTML5 та CSS3 та навчитись застосовувати бібліотеки Twitter Bootstrap щодо швидкої побудови Web-інтерфейсу користувача.

## Завдання
Розробити статичні Web-сторінки додатку «Калькулятор» та перевірити коректне їх відображення на екранах різного розміру. Програмувати функціональність сайту на цьому етапі не потрібно.

## Структура репозиторію
* `index.html` — головна робоча сторінка додатка (інтерфейс калькулятора з двома режимами, реалізованими через CSS Radio Hack)
* `login.html` — сторінка входу до системи (поля: email, пароль)
* `register.html` — сторінка реєстрації користувача (поля: ім'я, email, стать, дата народження)
* `profile.html` — сторінка профілю користувача з відображенням даних у табличному вигляді
* `about.html` — сторінка з емблемою та коротким описом додатка
* `README.md` — файл опису проєкту
